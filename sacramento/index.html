12 — Sources

Where This Data Comes From

CHP CCRS
California Highway Patrol publishes all crash records through the Crash Records System (CCRS) on data.ca.gov. 19,499 crashes in Sacramento County in 2024, each geocoded with coordinates, severity, and behavioral flags from the Parties detail table.

CCRS Dataset on data.ca.gov
13 — Methodology

How I Built This

Raw CCRS crash records are queried from the CKAN DataStore API, filtered to County Code 34 (Sacramento). Party-level data is joined by Collision ID to extract DUI/impairment status and speed limits. Records without valid coordinates are excluded. 19,499 records processed for 2024.

Data: CHP CCRS
